likeco-btn

在任何頁面插入「讚賞」按鈕

只要有讀者點一下「Like」按鈕，就能以實際行動支持原創作者。官方的 LikeCoin 按鈕僅支援 Medium 平臺及 WordPress 外掛；但現在您也可以用 likeco-btn 在任何頁面上插入讚賞按鈕。

步驟

  1. 透過下方的程式碼配置器，填寫您的「Liker ID」。
  2. (選填) 透過下方的程式碼配置器，填寫您的網頁網址。
  3. 複製該程式碼片段，並貼至目標網站的 HTML。
  4. 分享此頁面，一同開始賺取 LikeCoin！

程式碼配置器

顯示推廣 likeco-btn 的連結

有些固定網址包含 query 參數，如 WordPress 預設 permalink 即包含 ?p=xx 的參數。我們預設只保留 p 參數，若您需要保留更多參數，請在此設定（以半形逗號分隔）。

產生程式碼
步驟 1：將此段程式碼放在 </body> 結束之前 
<script src="https://noobtw.github.io/likeco-btn/likeco-btn.js"></script>
步驟 2：將此段程式碼放在想要插入 likeco-btn 的地方 
<div class="likeco-btn" data-name="noobtw7" data-url="<?php the_permalink(); ?>"></div>

常見問題

LikeCoin 是什麼？

詳細請參考 LikeCoin 官方網站。

Liker ID 在哪裡索取？

首先您需要註冊 LikeCoin 帳號。若您已註冊成功，可以在 like.co/in 的個人頁面找到您的 Liker ID

likeco-btn 和 LikeCoin 是什麼關係？我的讚賞會被抽成嗎？

我們和 LikeCoin 官方沒有任何關係。由於 LikeCoin 官方遲遲未推出可以放置在任何頁面的 LikeCoin 讚賞按鈕，因此我們製作了一個便於配置按鈕的小工具。使用 likeco-btn 完全免費，但您也可以考慮 捐獻 LikeCoin。

產生的 likeco-btn 程式碼要放在哪裡？

步驟 1 的程式碼請貼在 </body> 結束之前的位置；步驟 2 的程式碼請貼在你想要讚賞按鈕的地方，通常在文章的底部。例如如果你使用 WordPress 的話，可能會貼在 single.php；如果你使用 Ghost 的話，可能會貼在 post.hbs。
